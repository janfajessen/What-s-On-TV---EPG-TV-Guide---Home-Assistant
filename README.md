## рҹ”Қ Never miss what you want to watch

The keyword watch system lets you search across all configured EPG sources simultaneously. Some ideas:

- вҡҪ **Lost the match live and want to see?** Set a watch for `"LaLiga"`,`"Champions"`,`"Libertadores"`, or `"Premier League"` вҖ” you'll get notified before the replay airs, without any spoilers in the notification
- рҹҺ¬ **Looking for a classic you can never find?** Try `Funny games`, `"The Apartment"`,`Rope`, `"Casablanca"`, `"Blade Runner"`, `"The Godfather"` or `"2001"` вҖ” old films rotate on cable channels more often than you'd think
- рҹ“ә **Never miss your favourite show:** Set `"Jeopardy"`, `"MasterChef"`, `"Survivor"` or `"Who Wants to Be a Millionaire"` and get alerted before it starts
- рҹҺӨ **Music fan?** Search `Eminem`,`"Glastonbury"`, `"Eurovision"` or your favourite artist's name
- рҹҢҚ **Documentary hunter?** Try `"Planet Earth"`, `"Cosmos"` or `"Ken Burns"` вҖ” they rerun constantly on documentary channels
- рҹҸҺпёҸ **Sports in general:** `ATP1000`,`"Formula 1"`, `"La vuelta"`, `"NBA"` вҖ” works across all your configured countries simultaneously
- рҹҰҲ The ultimate channel surfing trap: Set `"Jaws"` вҖ” it's been airing every summer weekend since 1975 and somehow always catches you off guard
- рҹҺ„ Tired of missing the annual TV tradition? Set a watch for `"Home Alone"` вҖ” because it airs every Christmas on at least 12 channels simultaneously, and somehow you always miss it

> The **Word** match mode finds `"Friends"` but not `"Girlfriends"` or `"Unfriendly"`. The **Contains** mode is broader. Use **Exact** only for very specific titles.

A worldwide EPG (Electronic Programme Guide) integration for Home Assistant. Turns any XMLTV-compatible TV guide source into HA sensors вҖ” one sensor per channel, updated automatically.

## Features

- **115 countries** supported.
- One **sensor per channel** with current programme, next programme, progress percentage and full schedule
- **Keyword watches** вҖ” save a keyword and get notified when a matching programme appears in the guide, via any HA notification service
- Watches stored in **HA Storage** (persistent across restarts and devices)
- Supports **multiple instances** (different EPG sources simultaneously)
- Automatic refresh (configurable interval, 1вҖ“24 h)

## Installation

### Via HACS (recommended)

1. Open HACS вҶ’ **Integrations** вҶ’ вӢ® вҶ’ **Custom repositories**
2. Add `https://github.com/janfajessen/What-s-On-TV---EPG-TV-Guide/` вҖ” category **Integration**
3. Search for **What's On TV** and install
4. Restart Home Assistant

### Manual

Copy the `custom_components/whatson_tv/` folder to your HA `config/custom_components/` directory and restart.

---

## Configuration

Go to **Settings вҶ’ Devices & Services вҶ’ Add Integration вҶ’ What's On TV**.

**Step 1 вҖ” Source:** select your country/EPG source and update interval.

**Step 2 вҖ” Channels:** select the channels you want to track (the list is fetched live from the EPG source).

You can add multiple entries for different countries simultaneously.

---

## Sensor Attributes

Each channel sensor (`sensor.epg_<channel_name>`) exposes:

| Attribute | Description |
|---|---|
| `title` | Current programme title |
| `start` / `end` | Start / end time (ISO 8601 UTC) |
| `description` | Programme description |
| `image` | Programme artwork URL |
| `progress_pct` | Current programme progress (0вҖ“100) |
| `next_title` | Next programme title |
| `next_start` / `next_end` | Next programme times |
| `channel_id` | Channel ID in the EPG source |
| `channel_name` | Channel display name |
| `channel_icon` | Channel logo URL |
| `schedule` | List of upcoming programmes (up to 50) |

## Services

### `whatson_tv.add_watch`

```yaml
action: whatson_tv.add_watch
data:
  keyword: "Champions"
  notify_service: "notify.telegram_mybot"
  hours_before: 2   # optional, default 24
```

### `whatson_tv.remove_watch`

```yaml
action: whatson_tv.remove_watch
data:
  watch_id: "abc12345"
```

## Automations вҖ” Future Programme Notifications

You can replicate the Notify Card watch behaviour directly in HA automations:

```yaml
alias: "EPG вҖ” Notify when Champions League appears"
trigger:
  - platform: template
    value_template: >
      {% set keyword = "champions" %}
      {% set hours_before = 2 %}
      {% for state in states.sensor %}
        {% if state.attributes.channel_id is defined %}
          {% for prog in state.attributes.programacion | default([]) %}
            {% set start = prog.inicio | as_datetime %}
            {% set diff_h = ((start - now()).total_seconds() / 3600) %}
            {% if keyword in prog.titulo | lower and diff_h >= 0 and diff_h <= hours_before %}
              true
            {% endif %}
          {% endfor %}
        {% endif %}
      {% endfor %}
condition: []
action:
  - action: notify.telegram_mybot
    data:
      message: >
        {% for state in states.sensor %}
          {% if state.attributes.channel_id is defined %}
            {% for prog in state.attributes.programacion | default([]) %}
              {% set start = prog.inicio | as_datetime %}
              {% set diff_h = ((start - now()).total_seconds() / 3600) %}
              {% if "champions" in prog.titulo | lower and diff_h >= 0 and diff_h <= 2 %}
                рҹ“ә {{ state.attributes.channel_name }} вҖ” {{ prog.titulo }}
                рҹ•җ {{ start | as_local | as_timestamp | timestamp_custom('%H:%M') }}
              {% endif %}
            {% endfor %}
          {% endif %}
        {% endfor %}
```

> **Tip:** Use the **Notify Card** for a simpler no-code experience вҖ” it handles watches automatically without writing automations.
